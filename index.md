# 专业图片参考器 LookBack
类似 pureRef，面向美术人士，收集素材，美图使用

支持各种网站
相当于在浏览器上会多一个插件，右键元素，菜单中会多一个收藏
在收藏时，同时存储多个信息
用户点击按钮，触发收藏操作，后端进行 tag 打标后进行落库（找一下最佳实践，默认生成 tag，且 tag 可以用户自定义），所有信息存储为一个 meta，存储到本地特定文件夹

然后用户打开参考器时，会瀑布图式的显示所有收藏的图片（记得做懒加载与性能优化），用户可以根据 tag 进行搜索

同时左边是瀑布流，右边则是一个类似 pureRef 拖拉拽画布，你可以当做一个临时的图片 board，用户可以从左边拖拽图片，也可以自己在画布上添加图片

前端整体用 electron 技术栈
且需要支持 always on top 模式，用户可以将参考器固定在屏幕上，不被其他窗口遮挡
也要支持透明底模式，用户可以将参考器设置为透明，不遮挡其他窗口

产物有两个，一个是本地electron应用，用户可以在本地安装运行
另一个浏览器插件，用户可以在浏览器上使用
